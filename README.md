# Personal Portfolio Website – Data Visualization Assignment

## Purpose of the Website
This website is a personal portfolio page that presents information about my skills as a product designer and includes interactive data visualizations. The goal of the site is to demonstrate both design thinking and front-end development skills through visual communication and user interaction.

---

## Technologies Used
The website was built using:

- **HTML** – Structure of the website  
- **CSS** – Layout, styling, and responsive design  
- **JavaScript** – Interactivity and animation  
- **SVG** – Graphics for the visualizations  

---

## Website Navigation
The site contains two main pages:

### **Home Page**
- Displays profile information
- Shows background, skills, and professional focus

### **Visualizations Page**
Contains two interactive visualizations:

1. **Skills Bar Chart**  
   - An SVG-based bar chart showing a self-assessment of product design skills.
   - Each bar represents a skill level from 1–5.

2. **Interactive Bouncing Ball**  
   - A JavaScript animation demonstrating physics-based motion.
   - The green ball responds to gravity and user clicks.
